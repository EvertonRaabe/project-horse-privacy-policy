# project-horse-privacy-policy
Privacy Policy for Project Horse Manager mobile game.
📜 POLÍTICA DE PRIVACIDADE
Project Horse Manager

Última atualização: 02 de março de 2026

O aplicativo Project Horse Manager, desenvolvido por Tom Games, respeita a privacidade dos seus usuários. Esta Política de Privacidade explica como as informações são tratadas ao utilizar o aplicativo.

1. Coleta de Informações

O aplicativo não coleta diretamente dados pessoais como:

Nome

CPF

Endereço

E-mail

Contatos

Fotos

Localização precisa

No entanto, o aplicativo utiliza serviços de terceiros para exibição de anúncios.

2. Publicidade – Anúncios Recompensados

O aplicativo utiliza anúncios recompensados fornecidos pelo Google AdMob.

Os anúncios recompensados permitem que o usuário receba benefícios dentro do jogo ao optar por assistir a um anúncio voluntariamente.

O Google AdMob pode coletar automaticamente:

Identificador de publicidade do dispositivo (Advertising ID)

Endereço IP

Informações do dispositivo

Dados de interação com anúncios

Dados de uso do aplicativo

Essas informações são utilizadas para:

Exibir anúncios

Personalizar publicidade (quando aplicável)

Medir desempenho e evitar fraudes

Para mais informações, consulte a Política de Privacidade do Google:

https://policies.google.com/privacy

3. Armazenamento Local

O aplicativo armazena no próprio dispositivo apenas:

Progresso do jogo

Recursos virtuais

Histórico de corridas

Configurações internas

Esses dados não são enviados para servidores do desenvolvedor.

4. Consentimento e Controle de Anúncios

Dependendo da região do usuário (como União Europeia), poderá ser solicitado consentimento para anúncios personalizados.

O usuário pode redefinir ou limitar o identificador de publicidade nas configurações do próprio dispositivo Android.

5. Público-Alvo

O aplicativo não é direcionado especificamente a crianças menores de 13 anos.

Caso seja identificado que dados de crianças foram coletados por serviços de terceiros, medidas serão tomadas conforme exigido pelas políticas aplicáveis.

6. Compartilhamento de Informações

O desenvolvedor não vende, compartilha ou distribui dados pessoais dos usuários.

Dados eventualmente coletados pelo Google são tratados conforme a política de privacidade do Google.

7. Segurança

O aplicativo não coleta dados sensíveis.
Os serviços de publicidade utilizam medidas de segurança próprias para proteção de dados.

8. Alterações nesta Política

Esta Política poderá ser atualizada a qualquer momento.
Recomenda-se revisar esta página periodicamente.
