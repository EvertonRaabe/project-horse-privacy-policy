Política de Privacidade

Project Horse  ·  Terknaut Games

📅 Última atualização: 02 de março de 2026

1 Apresentação

O aplicativo Project Horse, desenvolvido por Terknaut Games, respeita a privacidade dos seus usuários. Esta Política de Privacidade explica como as informações são tratadas ao utilizar o aplicativo.

2 Coleta de Informações

O aplicativo não coleta diretamente dados pessoais como:

  • Nome
  • CPF
  • Endereço
  • E-mail
  • Contatos
  • Fotos
  • Localização precisa

No entanto, o aplicativo utiliza serviços de terceiros para exibição de anúncios.

3 Publicidade – Anúncios Recompensados

O aplicativo utiliza anúncios recompensados fornecidos pelo Google AdMob, que permitem ao usuário receber benefícios dentro do jogo ao assistir voluntariamente a um anúncio.

O Google AdMob pode coletar automaticamente:

  • Identificador de publicidade do dispositivo (Advertising ID)
  • Endereço IP
  • Informações do dispositivo
  • Dados de interação com anúncios
  • Dados de uso do aplicativo

Para mais informações, consulte a Política de Privacidade do Google →

4 Armazenamento Local

O aplicativo armazena somente no próprio dispositivo:

  • Progresso do jogo
  • Recursos virtuais (Ouro)
  • Histórico de corridas
  • Configurações internas

Esses dados não são enviados para servidores do desenvolvedor.

5 Consentimento e Controle

Dependendo da região do usuário (como União Europeia), poderá ser solicitado consentimento para anúncios personalizados.

O usuário pode redefinir ou limitar o identificador de publicidade nas Configurações do Android → Privacidade → Anúncios.

6 Público-Alvo

O aplicativo não é direcionado a crianças menores de 13 anos. Caso seja identificado que dados de crianças foram coletados por serviços de terceiros, medidas serão tomadas conforme exigido pelas políticas aplicáveis.

7 Compartilhamento de Informações

O desenvolvedor não vende, compartilha ou distribui dados pessoais dos usuários. Dados eventualmente coletados pelo Google são tratados conforme a política de privacidade do Google.

8 Segurança

O aplicativo não coleta dados sensíveis. Os serviços de publicidade utilizam medidas de segurança próprias para proteção dos dados coletados.

9 Alterações nesta Política

Esta Política poderá ser atualizada a qualquer momento. Recomenda-se revisar esta página periodicamente para se manter informado sobre eventuais mudanças.
